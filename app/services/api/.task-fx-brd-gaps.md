## Gap 2: Settlement Amount Formula (BRD 3.1.1)

Implement correct formula based on currency pair:

```
USD/VND         → amount × rate          (result in VND)
USD/... (base)  → amount ÷ rate          (result in USD)  
.../USD (quote) → amount × rate          (result in USD)
Cross pair      → amount × rate          (result in quote currency)
```

Create helper: `pkg/decimal/fx_calc.go`
```go
func CalculateSettlementAmount(notionalAmount decimal.Decimal, rate decimal.Decimal, pairCode string) (amount decimal.Decimal, currency string, error)
```

Rules:
- Determine if USD is base or quote from pair code
- If pair contains "/VND" → multiply, result in VND
- If pair starts with "USD/" → divide, result in USD
- If pair ends with "/USD" → multiply, result in USD  
- Otherwise (cross pair) → multiply, result in quote currency (part after /)
- Auto-calculate on create/update, store in settlement_amount + settlement_currency
